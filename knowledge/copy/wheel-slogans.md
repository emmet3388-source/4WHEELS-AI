# 4WHEELS AI · 鋁圈標語與情緒語彙庫

用途：
- 生成鋁圈 / 輪圈 / 改裝鋁圈 / 高級鋁圈內容時優先引用。
- 不要只講「帥」，要打到升級感、身份感、操控感、別人看你的眼神、車主品味、開出去的氣場。
- 適用於六平台貼文、Hooks、短影音開場、封面字卡、圖片 Prompt。

## 核心情緒

- 升級感
- 身份感
- 操控感
- 別人看你的眼神
- 車主的品味
- 開出去的氣場
- 細節完成度
- 整台車的靈魂

## 高級質感系

適合歐系車、精品感、高級鋁圈、IG / Facebook / YouTube。

- 真正懂車的人，先看輪圈。
- 車還沒到，氣場先到了。
- 改的不是輪圈，是整台車的靈魂。
- 一組輪圈，決定這台車的等級。
- 有些車很貴，但少了這組輪圈還是不夠完整。
- 細節，才是豪車真正的語言。
- 輪圈不只是配件，是品味。
- 低調的人，都改得很細。

## 性價比衝擊系

適合短影音 Hook、TikTok、Reels、LINE VOOM。

- 花最少，帥最多。
- 一組輪圈，直接像換一台車。
- CP值最高的改裝，永遠是輪圈。
- 不用改動力，光換輪圈就帥爆。
- 車不用賣，先換輪圈。
- 同一台車，換完像貴100萬。
- 改輪圈，是最容易讓人後悔改太晚的東西。

## 熱血操控系

適合性能車、改裝玩家、操控型輪圈內容。

- 每一次過彎，都在證明你的選擇。
- 輕一點，快很多。
- 真正的性能，從簧下重量開始。
- 速度可以很快，質感不能妥協。
- 輪圈，是性能的第一步。
- 不只是帥，是操控。

## 抖音 / Threads 爆款風

短句、有記憶點、有留言感。

- 帥，是停好車後還會回頭看。
- 車主最容易上癮的改裝，就是輪圈。
- 你以為差不多，懂的人一眼就知道。
- 男人的鞋子，叫輪圈。
- 輪圈一換，人生都順了。
- 先改輪圈的人，通常最懂車。
- 車有沒有靈魂，看輪圈就知道。

## 外匯胎 / 鋁圈店現實感文案

適合 4WHEELS 銷售語氣，要有現實感與花錢邏輯。

- 有些人花幾十萬改車，有些人先改對輪圈。
- 真正厲害的車主，都知道錢該花在哪。
- 改裝不一定最貴，但一定要最有感。
- 輪圈選對，整台車直接升級。
- 預算有限，更要改得有感。
- 先把氣場做出來。

## 車主族群版本

### Tesla / 電車族

- 安靜很重要，氣場更重要。
- 電車的第一個改裝，通常是輪圈。
- 續航要有，帥也不能少。

### BMW / 歐系車

- 德系的精髓，在細節。
- BMW 不改輪圈，總覺得少一味。

### Alphard / LM / 豪華商務車

- 老闆的車，氣場不能輸。
- 商務車的質感，從輪圈開始。

### 年輕族群

- 沒改輪圈，等於還沒開始玩車。
- 車可以慢慢改，但輪圈一定先。

## 使用規則

- 內容提到鋁圈、輪圈、輪框、J值、ET、PCD、CB、改裝鋁圈、高級鋁圈時，優先引用本語彙庫。
- 推薦鋁圈不能只寫好看，要至少帶到一個情緒：氣場、品味、升級感、操控感、身份感。
- 規格仍要保守：PCD、CB、J值、ET、卡鉗干涉、葉子板干涉不確定時標註待確認。
- 社群貼文可以把標語改寫成平台語氣，但不能失去原本的短句力道。
