4WHEELS AI 指令總表

專案位置

Terminal 先進入專案：

cd "/Volumes/Extreme SSD/4WHEELS-AI"

啟動 AI 系統

啟動主系統

npm run ai

功能：

  • 建立 / 更新車款資料庫
  • 六大平台內容生成
  • AI Agent 角色生成
  • 自動判斷車款

標題生成器

啟動標題生成

npm run title

功能：

  • Threads 爆文標題
  • TikTok Hook
  • YouTube 點擊標題
  • IG Reels 標題

常用 Terminal 指令

查看目前位置

pwd

查看資料夾內容

ls

開啟目前資料夾

open .

開啟 package.json

open package.json

AI 系統架構

車款資料庫

位置：

knowledge/cars/

用途：

  • 儲存各車款資料
  • 原廠輪胎規格
  • 載重指數
  • 車型特性

平台知識庫

位置：

knowledge/platforms/

包含：

instagram.txt
facebook.txt
youtube.txt
line-voom.txt
tiktok.txt
threads.txt

用途：

  • 記錄平台流量特性
  • 記錄受眾偏好
  • 記錄平台語氣

AI Agent 人格

位置：

knowledge/agents/

包含：

tire-master.txt
female-driver.txt
car-modifier.txt
social-editor.txt
short-video-director.txt

用途：

  • 控制 AI 語氣
  • 控制內容人格
  • 控制角色視角

系統記憶

位置：

包含：

brand.md

用途：

  • AI 品牌記憶

常用工作流程

建立新車款資料

npm run ai

選：

1

輸入：

Toyota RAV4

結果：

  • AI 自動查詢車款
  • 建立 knowledge/cars 資料

生成六平台內容

npm run ai

選：

2

輸入：

Luxgen N7輪胎怎麼選？

再選代理角色。

結果：

  • AI 自動判斷車款
  • 自動讀知識庫
  • 生成六平台內容

生成爆款標題

npm run title

輸入：

好Ｘ多輪胎真的划算嗎？

結果：

  • Threads 標題
  • TikTok Hook
  • YouTube 標題
  • IG Reels 標題
