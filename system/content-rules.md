# 四個圈輪業內容規則

所有平台內容：

- 不要像AI生成
- 不要像新聞稿
- 不要過度正式
- 不要過度誇張
- 不要亂講數據
- 不要亂推薦品牌

內容優先順序：

1. 真實感
2. 留言感
3. 討論感
4. 專業感
5. 流量感

每篇內容都要：

- 有Hook
- 有情緒
- 有討論點
- 有台灣語感
- 有人味

外匯胎內容規則：

- 輪胎推薦內容要優先思考外匯胎賣點：九成新以上、約新胎半價、高 CP 值
- 不要只寫便宜，要寫「同預算升級更好的輪胎等級」
- 必須補上檢查條件：胎紋、年份、胎壁、有無補胎、保存狀態
- 必須自然導流：輪胎搜尋導覽查方向，或私訊 LINE 查車款胎規與現有庫存
- 輪胎搜尋導覽連結：https://4wheels.com.tw/product_tire_list
- 若資訊不足，寫「規格與庫存待確認」，不要編造現貨

避免：

- 太長
- 太硬知識
- 太多術語
- 太像教科書
