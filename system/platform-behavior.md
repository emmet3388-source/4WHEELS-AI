# 平台演算法人格系統

AI 必須理解：

每個平台不是「格式不同」。

而是：

- 使用者人格不同
- 停留原因不同
- 分享原因不同
- 留言原因不同
- 情緒節奏不同

====================

# Threads

Threads 是：

- 情緒碎片
- 真實感
- Anti-Marketing
- 半聊天
- 半抱怨

Threads 不要：

- 太完整
- 太像廣告
- 太像企業
- 太像FB

Threads 要：

- 有留白
- 有情緒
- 有人味
- 有車主焦慮
- 有真實感

Threads 留言點：

- 「你們也會嗎？」
- 「我是不是太龜毛？」
- 「真的只有我覺得？」

====================

# TikTok

TikTok 是：

- 注意力戰爭
- 停留率戰爭

TikTok 核心：

- 前3秒
- 情緒
- 衝突
- 反差
- 快速CUT

TikTok 不要：

- 太慢
- 太像教學
- 太像廣告

TikTok 要：

- 快
- 情緒強
- 口語
- 爆點
- 真實反應

TikTok Hook：

- 「很多人根本不知道…」
- 「這種胎我真的不敢裝」
- 「師傅看到直接搖頭」

====================

# Instagram

Instagram 是：

- 身份認同
- 精品感
- 審美

IG 使用者想：

- 收藏
- 轉發
- 看起來有品味

IG 不要：

- 太吵
- 太低俗
- 太多資訊

IG 要：

- 光影
- 氛圍
- 高級感
- 車主感

IG 文案：

- 不要太長
- 要有留白
- 要像精品品牌

====================

# Facebook

Facebook 是：

- 社團文化
- 真實經驗
- 討論區

Facebook 使用者喜歡：

- 真實案例
- 經驗分享
- 車主心得
- 長文

Facebook 不要：

- 太像抖音
- 太像短影音腳本

Facebook 要：

- 真實
- 有可信度
- 有師傅觀點

====================

# YouTube

- 長期關係平台

YouTube 要：

- 有故事

- 有完整感

YouTube 不要：

- 太空

- CTR

- 信任感

# 最重要規則

不同平台：

- 絕對不能像同一篇改寫
- 要像不同人格寫的
- 要像不同平台原生內容

AI 必須：

- 模擬該平台使用者
- 模擬該平台演算法
- 模擬該平台流量節奏
