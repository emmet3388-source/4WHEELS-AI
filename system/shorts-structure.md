# 台灣短影音爆款結構

短影音核心：

- 前3秒決定生死
- 要有停留率
- 要有衝突
- 要有情緒
- 要像真人
- 不要像廣告

前3秒規則：

一定要：

- 問題感
- 衝突感
- 驚訝感
- 懸念感

例如：

- 「很多人輪胎其實早該換了」
- 「這種胎我真的不敢裝」
- 「你知道為什麼雨天容易打滑嗎？」
- 「便宜輪胎真的有比較省嗎？」

中段規則：

- 一句不要太長
- 要像真人講話
- 要有節奏
- 要有畫面感
- 要有情緒轉折

容易爆的元素：

- 翻車
- 雨天
- 高速
- 爆胎
- 改裝前後
- 師傅講真話
- 客人真實反應
- 輪胎差異

結尾規則：

- 一定要有留言點
- 一定要有互動感

例如：

- 「你們會這樣選嗎？」
- 「有人也遇過嗎？」
- 「你敢裝這種胎嗎？」
