# 4WHEELS AI · 導流與轉換規則

## 店家資訊

- 店名：4wheels 四個圈林口旗艦店
- 官方 LINE：https://page.line.me/4wheels
- 場內輪胎搜尋導覽：https://4wheels.com.tw/product_tire_list
- 地址：新北市林口區文化北路一段336號
- 營業時間：全年無休 10:00-22:00

## 導流目標

每篇內容至少要完成一個導流目標：

1. 留言：引導車主留下車款、年式、目前胎規或問題症狀。
2. 私訊 / LINE：引導傳照片、胎規、車款資料給官方 LINE。
3. 收藏：提供可保存的檢查清單、規格重點、選購方向。
4. 分享：製造「這篇要傳給朋友看」的真實理由。
5. 到店：在適合平台自然露出地址與營業時間。
6. 輪胎搜尋：輪胎 / 外匯胎 / 胎規內容可引導到場內輪胎搜尋導覽。

## 外匯胎導流主軸

四個圈主打外匯胎：

- 外匯胎九成新以上
- 價格約新胎半價
- 同樣預算，讓車主有機會升級更好的輪胎等級

導流時要讓車主知道：

- 先傳車款、年式、胎規到 LINE 查庫存
- 可以先到場內輪胎搜尋導覽查方向：https://4wheels.com.tw/product_tire_list
- 再確認胎紋深度、年份、胎壁、有無補胎、保存狀態
- 規格與價格依現場庫存確認，不要在貼文中寫死
- CTA 可以寫「想看你車款有沒有九成新外匯胎，丟車款跟胎規到 LINE」

## CTA 規則

不要只寫：

- 歡迎詢問
- 有需要可以私訊
- 依需求選擇

要寫具體動作：

- 留言你的車款 + 年式 + 目前胎規，我們幫你看適合方向。
- 傳輪胎側面照片到官方 LINE，我們先幫你看規格與年份。
- 先到輪胎搜尋導覽看你的胎規方向，再把車款 / 胎規傳到 LINE 確認現場庫存：https://4wheels.com.tw/product_tire_list
- 如果你是 RAV4 / CR-V / N7 車主，可以直接丟車款到 LINE，我們幫你抓原廠規格。
- 想確認會不會磨葉子板，請傳鋁圈 J 值、ET、胎規。

## 平台 CTA 差異

- TikTok：留言車款 / 私訊看規格 / 前 3 秒誘發留言。
- Instagram：收藏這篇 / 私訊車款 / 精品感但不要硬推。
- Threads：用碎片語氣引導留言，不要像官方客服。
- Facebook：引導留言討論與案例分享，可放完整店家資訊。
- YouTube Shorts：引導留言車款，補充完整檢查方向。
- LINE VOOM：口語短 CTA，適合直接導 LINE。

## 數據優化方向

- 停留率：前 1-3 句必須有痛點、反差、真相或車主焦慮。
- 留言率：每篇要有一個容易回答的問題。
- 收藏率：知識型內容要有清單、數字、規格、檢查步驟。
- 分享率：用「很多人不知道」「別被坑」「雨天/高速/家人安全」。
- 點擊率：CTA 要具體說明點 LINE 後能得到什麼。
